# 马年新春贺词生成应用 - 部署指南

## 项目结构
```
/
├── index.html          # 主页面
├── css/                # 样式文件
│   └── style.css       # 主样式文件
├── js/                 # JavaScript文件
│   ├── main.js         # 主逻辑
│   └── api.js          # API调用
├── assets/             # 静态资源
│   ├── images/         # 图片资源
│   └── animations/     # 动画资源
└── 部署指南.md          # 部署说明
```

## 部署步骤

### 1. 配置API密钥
打开 `js/api.js` 文件，修改以下配置：
```javascript
const API_CONFIG = {
    endpoint: 'https://ark.cn-beijing.volces.com/api/v3',
    apiKey: 'YOUR_API_KEY', // 替换为实际的API Key
    modelText: 'qwen3-max',
    modelImage: 'qwen-image-max'
};
```

### 2. 本地测试

#### 方法1：使用Python内置服务器
```bash
# 在项目根目录执行
python -m http.server 8000
# 或
python3 -m http.server 8000
```

然后在浏览器访问：`http://localhost:8000`

#### 方法2：使用其他本地服务器
- 使用 Node.js 的 `http-server`：`npm install -g http-server && http-server`
- 使用 PHP 内置服务器：`php -S localhost:8000`

### 3. 线上部署

#### 方法1：静态网站托管
- GitHub Pages
- Vercel
- Netlify
- 阿里云 OSS + CDN

#### 方法2：服务器部署
- 将项目文件上传到服务器
- 配置 Nginx/Apache 等Web服务器
- 确保正确设置 MIME 类型

## 使用说明

### 1. 访问应用
在手机浏览器中打开部署后的网址。

### 2. 生成贺词
- 点击「生成贺词」按钮
- 等待生成完成
- 查看生成的贺词

### 3. 操作选项
- **复制贺词**：点击后复制贺词到剪贴板
- **生成图片**：点击后生成并下载9:16比例的贺卡图片
- **重新生成**：点击后重新生成新的贺词

## 注意事项

1. **API密钥安全**：确保API密钥不被公开暴露
2. **网络连接**：应用需要网络连接才能使用AI生成功能
3. **浏览器兼容性**：建议使用现代浏览器（Chrome、Safari、Firefox等）
4. **图片生成**：生成图片可能需要较长时间，请耐心等待

## 故障排除

### 问题：无法生成贺词
- 检查网络连接
- 确认API密钥配置正确
- 查看浏览器控制台是否有错误信息

### 问题：图片下载失败
- 检查网络连接
- 确认浏览器权限设置
- 尝试重新生成图片

### 问题：页面显示异常
- 清除浏览器缓存
- 尝试使用其他浏览器
- 检查文件是否完整上传
